# Karat

## Usage ##
`man ./karat.1` for more details

## System specificiation (In Progress) ##
- Big Endian
- Multi-word instruction set

### Installation ###
`make vm ; make install`

With default modules
`make all ; make install`

### Preprocessor Flags ###
#### KDEBUG ####
Keep debug code in
#### KARAT\_PROD ####
Turn off some safety checks for production
